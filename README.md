# Juan Felipe Cerqueira Ladeira

Backend engineer specializing in Python, with a focus on building scalable APIs,
process automation, and AI-powered systems. I work across the full delivery
cycle from architecture and domain modeling to deployment and observability.

Currently building **DocAgent**: a multi-tenant AI agent platform with RAG over
PDFs, WhatsApp and Telegram integrations, and tool-calling agents orchestrated
with LangGraph. Everything runs locally on GPU via Ollama — zero external API
cost, privacy-first architecture.

I use **Claude Code** and **Gemini CLI** as core parts of my development workflow,
not just as helpers but as tools deeply integrated into how I architect,
refactor, and ship production code.

My engineering values: reproducible environments, rigorous testing, clear
documentation, and systems that are honest about their complexity.
